# 1️⃣ First enter MySQL from XAMPP
Run:
```bash
sudo /opt/lampp/bin/mysql -u root
```
If it asks password just press **Enter** (XAMPP root usually has no password).

# 2️⃣ Check if the user exists
Inside MySQL run:
```sql
SELECT user, host FROM mysql.user;
```
Look for:

```
campus_ai_user
```

---

# 3️⃣ If the user DOES NOT exist → create it

```sql
CREATE USER 'campus_ai_user'@'localhost' IDENTIFIED BY 'yourpassword';
```

---

# 4️⃣ Give the user access to the database

```sql
GRANT ALL PRIVILEGES ON campus_ai_db.* TO 'campus_ai_user'@'localhost';
FLUSH PRIVILEGES;
```

---

# 5️⃣ If the user exists but password is wrong

Reset it:

```sql
ALTER USER 'campus_ai_user'@'localhost' IDENTIFIED BY 'yourpassword';
FLUSH PRIVILEGES;
```

---

# 6️⃣ Check your PHP config file

Open:

```
/opt/lampp/htdocs/Admin-F/db.php
```

Make sure it matches:

```php
$conn = new mysqli(
    "localhost",
    "campus_ai_user",
    "yourpassword",
    "campus_ai_db"
);
```

---

# 7️⃣ Restart XAMPP

```bash
sudo /opt/lampp/lampp restart
```

---

# 🔎 Quick test

Open:

```
http://localhost/Admin-F/
```

---

💡 **Important tip (many people miss this):**

In **XAMPP**, sometimes the root user is easier during development:

```php
$conn = new mysqli("localhost","root","","campus_ai_db");
```

No password.
